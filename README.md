I build backend-first platforms with a simple philosophy: **sensitive logic stays on the server, clients stay thin.** My work spans two areas:

- 🖥️ **Infrastructure &amp; SaaS** — self-hostable PHP platforms, VPS automation, Telegram bots, and clean REST APIs. No heavy dependencies; ready for shared hosting or VPS.
- ⛓️ **Web3 &amp; Crypto** — smart contracts and a crypto-platform project (private), plus token and on-chain tooling.

Everything I ship for the public is open-source and bilingual (English / Arabic, full RTL).

## 🚀 Featured Project

### [Hetzner VPS Bot](https://github.com/q8ip/Hetzner-vps-Bot)
A complete PHP platform to **sell &amp; manage Hetzner Cloud VPS subscriptions**.

| Channel | What it does |
|---------|--------------|
| Web store | Browse plans, PayPal checkout, instant server activation |
| Admin panel | Clients, plans, stock, invoices, Hetzner, PayPal |
| Client portal | Telegram OTP login, server control, renewals |
| Telegram bot | Power on/off, reboot, rebuild, alerts |
| REST API v1 | JWT + OTP, ready for mobile apps |

> PHP 8.1+ · MySQL · zero Composer dependencies · EN/AR · **Open-source, launching soon 🔥**
